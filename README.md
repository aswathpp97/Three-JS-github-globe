# Three JS Github Globe

## Inspiration

This project was inspired by [Github's homepage](https://github.com/home)


## Usage

This project is bundled with [vite](https://vitejs.dev/)

```json
 "dev": "vite --host",
 "build": "vite build",
 "preview": "vite preview"
```
